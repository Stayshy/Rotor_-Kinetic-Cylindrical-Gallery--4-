    Перед запуском убедитесь:
    
    ☐ Docker установлен
      Проверка: docker --version
      Скачать: https://www.docker.com/products/docker-desktop
    
    ☐ Docker запущен
      Windows/Mac: Откройте Docker Desktop
      Linux: sudo systemctl start docker
    
    ☐ Порт 3000 свободен
      Windows: netstat -ano | findstr :3000
      Linux/Mac: lsof -i :3000
    
    ☐ Есть интернет (для загрузки образов)
    
    ☐ Минимум 2GB свободного места

│                      ⚠️  ЕСЛИ НЕ РАБОТАЕТ                                 │
└───────────────────────────────────────────────────────────────────────────┘

    Ошибка "Docker не найден":
    → Установите Docker Desktop

    Ошибка "Порт занят":
    → Измените порт в docker-compose.yml на 8080:80

    Ошибка при сборке:
    → docker system prune -a
    → docker-compose build --no-cache
    → docker-compose up -d

    Страница не загружается:
    → docker-compose logs -f (посмотрите что пошло не так)
    → docker-compose down && docker-compose up -d (перезапустите)

    Другие проблемы:
    → Откройте КАК_ЗАПУСТИТЬ.md - там есть решения

    Windows:        start-dev.bat
    Linux/macOS:    ./start-dev.sh
