        │         БЫСТРОЕ РЕШЕНИЕ ПРОБЛЕМ              │
        │                                               │
        ╰───────────────────────────────────────────────╯


    ❌ ПРОБЛЕМА: Порт 3000 занят
    ✅ РЕШЕНИЕ:  Измените в docker-compose.yml
                 "3000:80" на "8080:80"
                 Теперь: http://localhost:8080


    ❌ ПРОБЛЕМА: Docker не запущен
    ✅ РЕШЕНИЕ:  Windows/Mac: Запустите Docker Desktop
                 Linux: sudo systemctl start docker


    ❌ ПРОБЛЕМА: Ошибка при сборке
    ✅ РЕШЕНИЕ:  docker system prune -a
                 docker-compose build --no-cache


    ❌ ПРОБЛЕМА: Страница не загружается
    ✅ РЕШЕНИЕ:  docker-compose logs -f
                 docker-compose down
                 docker-compose up -d
