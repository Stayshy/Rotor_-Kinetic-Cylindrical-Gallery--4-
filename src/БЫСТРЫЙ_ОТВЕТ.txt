┌────────────────────────────────────────────────────────────────────┐
│  📊 ПРОЦЕСС СБОРКИ:                                                │
└────────────────────────────────────────────────────────────────────┘

  1. Docker скачает Node.js Alpine (~40MB)
  2. Установит npm зависимости (~300MB)
  3. Соберёт React приложение (vite build)
  4. Docker скачает Nginx Alpine (~10MB)
  5. Скопирует собранные файлы в Nginx
  6. Запустит контейнер на порту 80
  7. Пробросит порт 3000 → 80

  Порт занят:
  → Измените "3000:80" на "8080:80" в docker-compose.yml

  Docker не запущен:
  → Запустите Docker Desktop (Windows/Mac)
  → sudo systemctl start docker (Linux)

  Ошибка сборки:
  → docker system prune -a
  → docker-compose build --no-cache

  Страница не грузится:
  → docker-compose logs -f (посмотреть что не так)
  → docker-compose down && docker-compose up -d
